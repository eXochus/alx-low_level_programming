#This directory is about bit and bitwise manipulation

##Things to note:
>1. **Number bases:** Numbeer bases range typically from base two to any number you can think of
>However, the common ones known are
>>1. Base Two(Binary) 
>>2. Base Eight(Octal) 
>>3. Base Ten(Denary)
>>4. Base Sixteen(Hexadecimal) 

>2. **Conversions:** The number bases are convertible between one another with base ten serving as their intermediary 
>>   *Intermediary in the sense that, any bases can be converted directly to base 10 and vice versa but to convert from one base to another, it first has to be converted to base 10

>3. **Contents:** Each constituent of each base has to be 1 less than its base, *that is, a number in base 7 will only has its digits to be 6 and less. 

>4. **Notation:** The only notable notation is for the Hexadecimal, has it has already has all the figures exhausted, hence the usage of letters as follows
>    `A = 10` `B = 11` `C = 12` `D = 13` `E = 14` `F = 15`
>    *It should be noted that, b and 0x in front of figures signify base 2 and base 16 respectively
